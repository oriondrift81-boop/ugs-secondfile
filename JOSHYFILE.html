This page includes features of the site and how to use them

  • How to use: Click a game to launch it in a new window.
  • Stealth: Press ESC at any time to panic-redirect.
  • Favorites: Click the star to pin games to the top.
  • Deleting: Click the delete option, once activated any game you click on will be deleted
  • Undo: Click Undo button, this will undo all the games you have deleted, if a white screen shows up, close tab, and run code again
